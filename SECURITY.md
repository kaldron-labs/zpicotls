## Report a security issue

The zpicotls project team welcomes security reports and is committed to providing prompt attention to security issues. Security issues should be reported privately via h2o-vuln@googlegroups.com.

## Security advisories

Remediation of security vulnerabilities is prioritized by the project team. The project team endeavors to coordinate remediation with third-party stakeholders, and is committed to transparency in the disclosure process. The zpicotls team announces security issues via [GitHub release notes](https://github.com/kaldron-labs/zpicotls/releases) on a best-effort basis.
